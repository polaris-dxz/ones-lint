# no-js-in-ts-project

不推荐在项目中同时存在 `JS` 和 `TS` 文件。

## 规则内容

**错误示例**，TS 项目中包含 JS 文件:

```Bash
.
├── index.ts
├── home.js
└── tsconfig.json
```

 **正确示例**:

```Bash
.
├── index.ts
├── home.ts
└── tsconfig.json
```

## 规则选项

默认当存在 `commitlint.config.js`, `eslintrc.js`, `prettierrc.js`, `stylelintrc.js` 文件时不会报错，支持自定义设置文件白名单：

```json
"polaris/no-secret-info": [<enabled>, {
  'whiteList': [<string>],
  // 是否 merge 默认文件配置，默认为 true
  'autoMerge': <boolean>
}]
```
